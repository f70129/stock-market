台指期貨 & 台灣加權指數 費氏數列分析看板

## 功能

- **費氏數列 × 移動平均** 自動波段偵測、回檔/延伸位階、共振分析、明日目標價。
- **🚨 策略監控提醒看板**：依下列規則即時監控做多/做空訊號，並提供聲音與桌面通知提醒。

### 做多策略（順應長線多頭）
- 進場：收盤向上突破 **25MA** 且站上 **34MA**（過濾假突破雜訊）。
- 停損：跌破 **5MA**（短線動能耗盡提前撤出）。
- 終極移動停利：**80MA**（捕捉多頭肥尾行情）。
- 輔助：帶量跳空缺口「三日法則」、7–9 月除權息校正（改看加權大盤）。

### 做空策略（嚴格限制於熊市）
- 宏觀濾網（必要條件）：收盤位於 **200MA** 之下才允許做空。
- 進場：跌破 **15MA** 且位於 **20MA** 之下。
- 停損：站上 **8MA**（極端盤勢縮至 3MA）。
- 停利：偏離 **30MA** 出現大幅負乖離（緩漲急跌，提防軋空）。
- 輔助：KD 連續三根 K < 20「低檔鈍化」確認強空。

### 通用動態風控（ATR）
- 初始防守：單筆最大虧損控制在當日 **ATR 的 20%–25%** 距離內。
- 階梯式移動停利：停利線設於距現價 **ATR 的 33%–50%**，隨行情推進上移保本。

> ⚠️ 所有訊號僅供參考，不構成投資建議。市場有風險，操作請謹慎。
